## **Bloody-Escape (Game)**

Bloody Escape is a Python text-based escape room game where the player wakes up in a mysterious forest.

Solve puzzles, uncover secrets, and try to escape the eerie environment.



**-------------------------------------------------------------------------------------------------------------**



\## **HOW TO RUN**



1\. Make sure you have Python (latest version) installed.



2\. Download the project files.



3\. Open the main script in Python and run it.



**-------------------------------------------------------------------------------------------------------------**



\## **ABOUT**



-Created as a school project.



-Includes puzzles, mysteries, and hidden clues.



-Set in a dark and creepy atmosphere.



-Designed for fun while learning Python programming.
